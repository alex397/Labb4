Package lab4.data

The package contains files/classes GameGrid and GomokuGameState. The package represents the model of the game. I.e all data within a game is stored here and all functionalities (buttons, grid interaction etc) is defined here. The package is build by using a object of GameGrid (describes a board and possible actions important for the game like player movement) inside GomokuGameState (describes a game in progress by using functionalities in GameGrid depending on the state of the game) in order to make a complete model of a game.
